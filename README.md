# bukelilun

个人播客学习笔记。收听讲座、对谈类播客后，按固定工作流将内容提取为结构化的 Q&A 笔记，保留核心论点与待深挖线索。

## 目录结构

```
├── WORKFLOW.md          # 笔记提取工作流模板
├── raw/                 # 原始转写文本（语音转文字稿）
│   ├── 13_xxx.txt
│   └── ex_01_xxx.txt
├── 13_xxx.md            # 提取后的笔记
└── ex_01_xxx.md
```

- `raw/` — 讲座音频的转写原文，作为提取的输入素材
- 根目录 `.md` — 按 WORKFLOW 提取后的笔记成品，格式为模块化 Q&A + What's Left 线索区

## 编号规则

- 数字前缀（`13_`、`37_`）— 对应《不可理论》播客期数
- `ex_` 前缀 — 播客之外的扩展内容

## Commit 格式

```
<一句话概述>

[Notes]     ← 笔记增删改
[Raw]       ← 原始素材增删
[Chore]     ← 工程及结构相关
[Format]    ← 内容格式/样式调整
```

按需使用相关分类，无变更的分类省略不写。

范例：

```
初始化讲座笔记仓库

[Notes]
1.add 13_不可理论×独立日：语言学肄业生的一次谈话
2.add 37_茫茫相似将万物勾连
3.add ex_01_批判理论的文化边界：从欧陆到东亚社会

[Raw]
1.add 13_不可理论×独立日：语言学肄业生的一次谈话
2.add 37_茫茫相似将万物勾连
3.add ex_01_批判理论的文化边界：从欧陆到东亚社会

[Chore]
1.add WORKFLOW.md 讲座笔记提取模板
2.add README.md
```
