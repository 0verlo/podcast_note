# podcast note

个人学习笔记。收听讲座、对谈类音频节目后，按固定工作流将内容提取为结构化的 Q&A 笔记，保留核心论点与待深挖线索。

## 目录结构

```
├── WORKFLOW.md              # 笔记提取工作流模板
├── <节目名>/                # 按节目来源分目录
│   ├── raw/                 # 该节目的原始转写文本
│   │   └── 01_xxx.txt
│   └── 01_xxx.md            # 提取后的笔记
```

- 每个节目来源独立一个目录
- `<节目名>/raw/` — 音频转写原文，作为提取的输入素材
- `<节目名>/*.md` — 按 WORKFLOW 提取后的笔记成品，格式为模块化 Q&A + What's Left 线索区
