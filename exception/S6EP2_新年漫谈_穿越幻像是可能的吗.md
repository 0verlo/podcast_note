# 讲座笔记：新年漫谈——穿越幻像是可能的吗？
> 来源：《例外状态》S6EP2。主播诡橘子与嘉宾佳泰（欧洲高等学院博士生，师从齐泽克，研究巴迪欧）对谈。

---

## 模块一：利奥塔的《力比多经济学》——精神分析还是经济学？

**Q:** 佳泰即将翻译出版的利奥塔《力比多经济学》是一本什么样的书？
**A:** 利奥塔在这本书中将力比多的运动和流转作为研究对象，但并未遵循精神分析的叙事逻辑。他以法国当代形式主义研究的传统为框架，将力比多的流动、投注、升华、下降等过程处理为一种纯粹抽象的形式主义工作，并以"经济学"为类比——因为经济学本身就是一种冷酷无情的运算，力比多的各种冲动在互相抵消后也呈现出类似经济交换的中性形态。

**Q:** 利奥塔使用"力比多"这个概念，和精神分析有什么关系？
**A:** 利奥塔本人不认为这是精神分析。他仅借用了"力比多"这一个关键概念，并没有遵循精神分析关于欲望的解析性工作的思路。佳泰指出，利奥塔运用力比多的方式实际上接近拉康在《精神分析伦理学》研讨班中所批评的"欲望的自然主义解放"，与拉康走的是相反的道路。

**Q:** 利奥塔如何用经济学隐喻说明力比多的运作？
**A:** 以殖民时代的白银货币经济体系为例：一个水手在码头将一枚银币交给酒馆老板，这枚银币表面上只是一个中性的货币符号，但它实际浓缩了水手在海上数年搏命生涯中全部的力比多活动。通常被视为死物的象征物，恰恰是一系列力比多运动互相抵消后凝结的结果。强烈的冲动不可避免地在某个相互抵消的时刻转化为中性之物，并以这种中性形态完成交换——诡橘子将此概括为进入一种"equilibrium"（均衡）状态。

**Q:** 利奥塔在法国哲学中处于什么脉络？
**A:** 佳泰认为利奥塔更接近法国近代将爱欲（eros）当做本体论核心的这一流派，类似于让-吕克·马里翁的情欲现象学路径，而非精神分析范式。这也可以接续到巴特勒《欲望主体》所梳理的传统——整个20世纪法国哲学将爱欲或欲望置于了极高的本体论地位。国内对利奥塔的认知基本停留在"宏大叙事""后现代状况"，远远不够，他在形式主义化精神分析冲动方面的工作（不同于巴迪欧的数学化路径）尚未被充分关注。

## 模块二：幻想的结构性功能——为什么人必须有幻想？

**Q:** 人该不该有幻想（fantasy）？
**A:** 佳泰认为结论是"总得有"。如果一个人刀刃向内地把自己的幻想全部阉割掉，会陷入一种极度虚无主义的抑郁状态，甚至连起床刷牙都不想。幻想在一定程度上是必要的。

**Q:** 从纯粹形式主义的角度看，幻想为什么是必要的？
**A:** 回到拉康的理论：象征界中的符号是形而上学的最小形式，无论多小都不影响其意义，因此符号本质上是离散的、不接续的。幻想/想象的功能，就是通过某种"假定的或不充分的形式"，让这些本来无法接续的离散符号在某些局部呈现出连续运作的特征。这是想象能够让事物"运行起来"的根本的、机制性的原因。

**Q:** 沉浸在幻想中、被幻想裹挟会怎样？——以包法利夫人为例
**A:** 以包法利夫人或福楼拜《情感教育》中的弗雷德里克为例，这类人把虚构当了真。佳泰从个人经历出发指出，这种状态的核心问题在于"不可持续性"：幻想所支撑的力比多投注机制无法仅靠自身完成再生产和循环，它必然在某个时刻崩溃，且不以任何人的意志为转移。这不是包容或善意能解决的问题，而是结构性的不可持续。

**Q:** "幻想是不彻底的真实"是什么意思？
**A:** 不存在一种独立于唯物主义世界之外的、可以与真实剥离的幻想。幻想之所以成为幻想，唯一的原因是它"不彻底"——它悬停在一种真实和另一种真实的中间阶段，既不成为第一种真实，也不成为第二种真实。佳泰因此反对"顺其自然""想不通就不想"的生活态度：但凡有人采取这种态度，其力比多运行机制产生的责任一定在潜意识层面被转嫁给了他人，双方在潜意识中互不知情，但力比多的转移已经完成，所有人都因此变得暴躁。最低的要求是：不要把事情悬在中间，至少让它成为一个什么东西，而不是什么都不是。

## 模块三：穿越幻象的教育困境——觉醒还是犬儒？

**Q:** "穿越幻象"在美学与艺术社会学之间意味着什么？
**A:** 诡橘子提出美学与艺术社会学的根本区别：美学是消费者的美学，从康德出发讨论纯粹感受性、美感、接受，但不处理艺术的生产机制、生产方式和生产关系；艺术社会学是生产者的美学。佳泰补充：单纯的消费绝对不可持续，但纯粹的生产是可以可持续的（部落社会、农业社会已证明）。运用幻象的最好状态是：在局部让幻象发挥连接两个象征的效用，但在整体上知道它是一个幻象。

**Q:** 该不该在大学阶段就教学生"穿越幻象"？
**A:** 佳泰从精神分析临床经验出发认为要审慎。很多刚上大学的学生的欲望长期被压抑（一路考试考上来），他们的幻想更像是一种恶性压抑的代偿效应。可以给他们一些征兆或预示，但不要直接动手切割他们的幻想——直接剥夺幻想是一件很危险的事情。但诡橘子从当代艺术从业者的马克思主义立场出发认为，应该尽早揭露艺术生产过程中的矛盾和弊端，而不是让美术馆继续让人"用爱发电"。

**Q:** 批判理论对年轻学生可能造成什么负面效果？
**A:** 两个层面的实例。第一，学了文学理论后对阅读文本失去兴趣，总想用理论阐释，破坏了阅读本身的机制。第二，读了批判理论后觉得现实崩塌、信仰的符号体系动摇，直接抑郁。诡橘子举了一个陕师大文学系学生的例子：本科时被陈越老师讲的阿尔都塞、福柯等后现代理论"敲蒙了"，觉得一切被解构，研究生转向古代文学/古文字学，想寻找确定性。

**Q:** 如何避免批判沦为犬儒主义？
**A:** 佳泰指出，很多人被批判理论"敲了一棒子"后不是被"敲醒"了，而是被"敲蒙"了。他们没有学会别的，只学会了拿棒子去敲别人，陷入现代犬儒主义：变得刻薄、好面子、喜欢批判，同时又被摧毁了某些价值——青春期力比多的旺盛与批判工具的粗暴结合，产生一种既尖锐又空洞的状态。佳泰认为，真正的出路是进入公共领域：不能因为当下的公共领域存在庸俗的普遍性就逃避它，而应该用真正的普遍性去取代庸俗的普遍性。与公共领域的关系具有主奴辩证法的结构——"你不拥抱他对你的奴役，你就不能成为他的主人"。

## 模块四：欧洲高等学院与当代理论的代际困境

**Q:** 欧洲高等学院（EGS）是怎样一所学校？
**A:** EGS诞生于五月风暴之后，一群左翼哲学家无法在法国传统大学体制中立足，凑钱拉赞助自建的学校。它不授课，只举办研讨会（seminar），延续五月风暴式的对话与去思辨传统。校址选在瑞士瓦莱州萨斯费（Saas-Fee），阿尔卑斯山上一个偏僻小镇（同时也是高端滑雪度假地），没有城市生活，一帮哲学家围着小木屋和火炉铸金思。

**Q:** EGS目前的核心师资有哪些？
**A:** 老牌阵容：齐泽克仍在任教；巴迪欧已退休（曾设有勒内·笛卡尔教席）；朗西埃已退休；阿甘本仍在；奈格里和大卫·林奇均已去世；朱迪斯·巴特勒是常客。新生代：Avital Ronell（德里达弟子，偏文学研究方向）、Frank Ruda（齐泽克合作者，与齐泽克合著了*Towards a Liberal Fascism*）。校长Christopher Fynsk是布朗肖的学生，继承了布朗肖研究的主要学术班底。

**Q:** 齐泽克近年在EGS讨论什么？
**A:** 齐泽克每年在EGS有两到三期重点研讨班，近年主题包括*Towards a Liberal Fascism*。佳泰指出，齐泽克对当下"后后现代主义"状态的把握和反思，与我们在旧译著中读到的"复读"很不一样，他将当代世界的停滞状态形容为"像死亡的腐诗一样，毫无任何差异、毫无任何变化"。

**Q:** 朱迪斯·巴特勒近年的立场有什么变化？
**A:** 佳泰感受到巴特勒在EGS讲课的风格与其过去论著有明显变化。过去巴特勒在性别、政治少数族裔、犹太视角等交叉议题上持相对温和的立场，认为矛盾可以调和。但近年开始讲一些"非常残酷的东西"，表达出矛盾越来越不可调和的感受——尤其是在美国被举报之后，"实在界的创伤"真正到来了。

**Q:** 当代哲学面临怎样的代际困境？
**A:** 齐泽克、阿甘本、巴特勒等老牌哲学家仍在撑着EGS的牌面，但年龄都大了（齐泽克已75-76岁）。新一代学者虽然优秀且师承名门，但尚未形成具有穿透力的概念或本体论体系，更像是研究者而非"现象级大哲学家"。佳泰和诡橘子都认为：当前大部分理论已追不上时代，需要继续吸收转化60年代之后的理论、搞清其语境和生成过程，同时发展自己的问题意识，在新的历史事件出现时做出回应。全球化时代过于平静，新一波需要回应的猛烈现实也许在未来五到二十年内会出现。
