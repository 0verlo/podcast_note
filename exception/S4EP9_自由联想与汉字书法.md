# 讲座笔记：自由联想与汉字书法
> 来源：《例外状态》S4EP9。主播小柏与嘉宾施思（巴黎大学精神分析与精神病理学博士生）联合报告并互相追问。
> 注：原始转写中部分外文术语、书名和人名有明显噪音，以下按可确认的论点与脉络提炼。

---

## 模块一：自由联想的客观性——从不选择到能指自动性

**Q:** 拉康为什么把自由联想看成精神分析客观性的起点？
**A:** 因为精神分析若想把握心理反应的“自身现实性”，就不能从挑选哪些内容重要开始，而必须从“不再做出选择”开始。梦境、妄想、失误等都不能先验地被排除。对分析者来说，这意味着可以想到什么就说什么；对分析师来说，则意味着用悬浮注意去听取一切联想，而不是像暗示疗法那样预设某种方向。客观性首先因此表现为：把主体全部经验都纳入考察。

**Q:** 拉康如何批判联想主义心理学对心理结构的描述？
**A:** 施思指出，拉康认为联想主义心理学虽然自称科学，却并不客观。它只把“正常”意识状态当作核心对象，把梦、妄想、口误等异常经验视为可由抽象决定论处理的负现象；同时，它又试图用因果律、相似性等原则解释观念之间的联系。拉康认为，这种理论虽然触及了意识底层的被动性，却无法真正说明这种被动联系的结构，也缺少对主体全部经验的结构性解释。

**Q:** 能指自动性与精神分析式的现象学分析，怎样接上自由联想？
**A:** 拉康后来把联想中的被动联系重新界定为能指自动性：在分析中反复出现的东西，不是主体主动“记得”的，而是被能指链记住的；无意识中保存着一种主体并不掌握的知识。同时，分析又不能只做结构演算，还要在经验中捕捉意向与形象的显现，例如分析者总在朝向某个对象说话，也会把家庭成员或早年他者的形象叠加到分析师身上。这样，转移关系和迫害性的石化联系才会显露出来，分析的目标也不是宣布决定论，而是借由揭示这些联系，为主体打开新的可能性。

## 模块二：孟子与“性”——拉康为什么反对唯一解释

**Q:** 拉康为什么会借孟子“天下之言性也”来谈精神分析，而不是给出一个标准译解？
**A:** 小柏指出，拉康感兴趣的不是替孟子定于一尊，而是借这句话暴露不同解释背后的理论立场。他参考了 I. A. Richards 关于孟子的研究，但更在意的是 Richards 所代表的逻辑实证主义倾向：要求文本必须提供一个可把握的明确意义，否则就会被贬低乃至排除。拉康恰恰反对这种处理，因为一段话里那些看似不合逻辑、无法完全说明的部分，往往正是继续追索话语真相的线索。

**Q:** 这和精神分析的认识论有什么关系？
**A:** 在讲者看来，精神分析并不把解释理解成“找到唯一正确答案”。梦的解析不是符号词典，话语分析也不是把所有元素一一对应起来的翻译学。因为只要追索意义，最终总会遇到无法被完全说清的空缺；而用来解释话语的那套语言，本身也只是另一层隐喻，并不存在一个高于一切语言的“源语言”或元语言。精神分析因此不是去消灭含混，而是在含混和缺口处工作。

**Q:** 孟子的“性”与拉康的“没有性关系”之间，讲者看到了什么共同点？
**A:** 小柏提出了一种故意不正统、但能逼近精神分析真相的读法：中文里的“性”既可指天性、人性，也会牵出性别、sexuality 等含义。正因为这个词天然多义，它反而更能显示语言如何围绕一个无法被彻底固定的核心打转。精神分析不是给“性”提供一本质说明书，而是揭示主体说话时总在围着某个空缺、某个说不尽的东西反复逼近。

## 模块三：书法、主人话语与主体劈裂

**Q:** 如果精神分析不以口头语言中的固定意义为准，它靠什么保持严谨？
**A:** 小柏的回答是：严谨性未必首先存在于口头语言里，而可能更多存在于书写里。拉康会把某些无法用日常意义说清的话语，当作像数学符号那样严密的书写来处理；在这个意义上，他把“天下之言性也”读成主人话语的书写。主体总是被一个能指表征给另一个能指，在进入能指链时又有一部分东西掉落出来，这也就是他借主人话语与客体小 a 所强调的结构。

**Q:** 小柏为什么把孟子这句话与剩余享乐联系起来？
**A:** 她提到，拉康把“以利为本”读出了一种与资本主义时代相连的意味，并借此联想到马克思的剩余价值。对拉康来说，主体进入大他者秩序后，也会有某种东西被剥夺出去；剩余享乐正是一种对这种剥夺的结构性补偿。因此，这句话之所以重要，不是因为它定义了“性”的本质，而是因为它折射出语言本身如何被欲望驱动，并围绕一个空位生产出剩余。

**Q:** 为什么汉字书法能成为主体显现的一种方式？
**A:** 讲者把“字如其人”重新解释为一种精神分析命题。儿童练习书法时，身体姿势、执笔方式、规范要求与驱力身体之间持续拉扯，身体的享乐并不是直接表达出来，而是被压入书写痕迹之中。于是书法呈现给人的，不只是写了什么内容，而是主体如何在自己的独特性与大他者要求之间维持平衡。也正因此，书法让主体的劈裂状态变得可见：一半服从规范，一半又在笔势和痕迹里泄露出来。

## 模块四：草书、字符与艺术——从意义崩塌到客体显现

**Q:** 拉康为什么会特别强调汉字草书，而不是一般意义上的个人笔迹？
**A:** 施思的理解是，关键不只是“字写得有个人风格”，而是汉字本身带着公共可读的意义形式。草书通过省略笔画、扭曲形态，把这个公共形式击碎了。阅读者原本能“认出”它的意义，但到了草书面前，常常只能辨认出一个图像、一种风格、一股张力。也就是说，公共意义被悬置之后，手的独特性反而更强烈地显现出来。

**Q:** 这种从“认识字”到“只剩识别”的变化，对精神分析意味着什么？
**A:** 当字不再按日常用途发挥功能，语词会像物一样突兀地出现在人面前。施思把这一点和分析中的切分联系起来：分析师通过在某个能指上切断、停留，使它暂时脱离日常语义链，作为字面、作为声音、作为物被经验到。正是在这种意义屏障被打破的时刻，主体才有可能从中识别出某些与客体小 a、与焦虑相关的东西。

**Q:** 拉康后期把字符/书写与能指/言语区分开来，想说明什么？
**A:** 施思认为，拉康后期更强调字符或书写靠近实在的一面。能指之间的联系还多少可以解释，但到了字母、字符、书写痕迹这一层，意义就退后了，剩下的是更接近个体独特性、也更难被象征化的维度。她举汉字形声字为例：其中表音成分在新字里往往被剥离原本意义，只剩下一点声音性的残余。这种”只剩声音”的部分，恰好显示出一种离意义更远、离享乐和实在更近的结构。小柏接着指出，这一从早期到晚期的理论重心变化，也标志着拉康从”以父之名”的单数走向复数，从症状走向圣状的转向。

**Q:** 弗洛伊德和拉康对艺术的理解，差别在哪里？
**A:** 小柏把问题提得很尖锐：弗洛伊德讲升华，更像把趋力导向文明化、可被社会承认的对象；拉康谈艺术，尤其谈草书、绘画、凝视时，更强调它如何把客体提升到原物的位置，使某种超出公共用途的东西显现出来。施思认为两者确有方向上的区别：前者更像从实在向象征位移，后者更像从象征朝实在逼近。但她也补充，二者并非全然对立，因为艺术都可能承担重新安置享乐的功能。到了拉康后期，艺术甚至可以作为圣状，为某些精神病结构的主体提供稳定化和重新定位享乐的支点；小柏引米勒的 l'escabeau（小台阶/梯凳）概念补充说，美是挡在原物之前的一道栅栏，精神病主体通过圣状重新定位享乐时，往往仍需经过某种被社会承认的美学支撑；施思则引乔伊斯为例呼应，指出作为作家的乔伊斯也必须提供一种想象性的认同和稳定化功能，在这一层面上拉康与弗洛伊德是相近的。

## 模块五：临床解释的作用——松动能指，把焦虑化为欲望

**Q:** 精神分析式的解释，为什么不是给病人一个更准确的说明？
**A:** 小柏给出的回答非常临床化：解释的目的不是关闭意义，而是松动意义。她举了两个都围绕“黑”展开的案例。一个政治难民在遭遇监禁后反复出现黑暗、老鼠、石头等幻觉，但这类体验之所以对他如此迫害，并不只是因为后来的创伤，更因为童年早已有“不听话就把你扔进黑井里”的母亲话语。另一个遭受种族歧视的黑人病人，关于“黑”的痛苦也不仅来自社会羞辱，还叠加了儿时被关进小黑屋的经验。分析要处理的正是这种早期标记能指如何把主体石化，并在后来经验中反复启动。

**Q:** 这些案例说明了怎样的解释原则？
**A:** 它们说明，解释不是替主体归纳一个结论，而是让某个原本只会导向迫害场景的能指，重新和别的能指发生联系。施思也呼应说，拉康在第11研讨班强调过，分析式解释是“打开一些东西”，而不是像学究式解释那样，通过确定一个意义来把话语封死。若某个主人能指始终自我重复、无法滑向别的能指，主体就会被困在同一场景里；分析师的介入则是松土，让它重新开始移动。

**Q:** 那个“星球/明星”的案例，具体体现了什么？
**A:** 小柏提到一位妄想症患者总说自己住在某个“star”上。到分析后期，医生没有追问这个愿望究竟源自谁，而是在病人说自己喜欢弹吉他、想上台表演时，顺势把“住在一颗星球上”引向“成为一颗明星”。解释的效力恰恰在这里：不是消灭这个词，而是让它从单一妄想意义中松开，露出另一种主体化的可能性，让主体不必永远被同一个能指迫害。

**Q:** 对那部分无法解释、无法还原的身体性享乐，分析最后怎么处理？
**A:** 两位讲者的答案都指向欲望化。进入分析时，主体常带着过量享乐、焦虑和对他者欲望的高度防御；而当分析师作为第三方介入，原先僵死的能指系统开始松动，焦虑也可能从纯粹的受难状态转化为欲望的通道。小柏把这一过程概括为：一方面，从完全受制于大他者欲望，挪出一点自己的欲望；另一方面，把那种无法解释的身体性享乐安置到满足与不满足的交替之中，使主体能够承受它，而不是被它完全淹没。
