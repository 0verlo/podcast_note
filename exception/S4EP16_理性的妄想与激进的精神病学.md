# 讲座笔记：理性的妄想与激进的精神病学
> 来源：《例外状态》S4EP16。主讲：小柏、粉色海猡；评论与答问：陈劲骁。
> 注：原始转写噪音较重，以下人名、术语与机构名按上下文统一为雅斯贝尔斯、施瑞伯、拉博德、托斯凯勒等可确认写法。

## 模块一：雅斯贝尔斯的“不可理解性”与精神分析的分歧

**Q:** 雅斯贝尔斯怎样区分原发性妄想与继发性妄想？
**A:** 小柏先回到《普通精神病学》。在雅斯贝尔斯那里，原发性妄想是“突然降临”的：它没有可追溯的心理前因，不靠推理得出，病人却对其有强烈确信，并常伴随一种难以描述的氛围性转变；继发性妄想则还能沿着人格、生活史和情感连续性去理解，比如嫉妒、受挫、屈辱等现实材料如何被卷入妄想。于是“可理解/不可理解”就成了区分病理空间的首要标准。

**Q:** 这种区分为什么会持续影响当代精神病学？
**A:** 因为一旦把某类症状先行判定为“不可理解”，医生就会自然转向因果解释，例如生理、基因、脑功能退化等路径。小柏指出，DSM 对精神分裂的怪异性、第一等级症状的强调，某种程度上仍继承了这一逻辑。这样做会带来三个后果：第一，不可理解性既被当成病因又被当成病果；第二，精神病被放到与“正常”质性断裂的位置，压缩了共情与连续性理解的空间；第三，诊断高度依赖临床者的主观门槛，医生一旦觉得“我不懂”，阅读症状的工作就会提前终止。

**Q:** 小柏为什么说精神分析不会停在“不可理解”这一步？
**A:** 因为在弗洛伊德和拉康那里，理解不是放弃，而是通过无意识的逻辑让症状重新变得可读。精神分析并不把妄想简单还原为理性失灵，也不把精神病主体直接交给生物决定论，而是追问：主体究竟是以什么样的结构、什么样的语言关系进入世界的。小柏因此把萨斯的问题意识概括为：如何避免把精神病当作纯生物退化，同时又不落回简单共情。

## 模块二：萨斯的维特根斯坦路径——为何能解释施瑞伯，却解释不了欲望

**Q:** 萨斯为什么会借维特根斯坦的唯我论批判来读施瑞伯？
**A:** 小柏概括说，萨斯想避免那种“我无法共情，所以你是疯的”诊断方式，转而从哲学的抽象层面把握妄想的抽象。施瑞伯式妄想在他那里像是一种准唯我论：主体把自己说成世界中心、全部经验的承载者，但维特根斯坦会指出，这种说法并没有创造出新的世界内容，只是在语言里重复“我”的位置；而且它一旦被说出来，就已经预设了听者与外部检验机制，从而违背了真正的唯我论前提。

**Q:** 小柏认同了萨斯的哪一层推进？
**A:** 她认同萨斯没有把疯狂视作理性的缺席。无论是萨斯还是拉康，都试图反对“精神病=纯粹退化”的看法。小柏特别借拉康的早期文本强调：疯狂不是偶然的生理故障，而是人之存在内部始终张开的可能性；它不是和“正常人性”毫无关联的异类，而是理性自身内部的一种极限经验。

**Q:** 但她为什么又说萨斯和维特根斯坦都踩到了同一块礁石？
**A:** 因为他们都默认语言共同体有一个稳定、完备的一致性框架，好像大他者始终在那里保证意义与判断的可通约。小柏认为，这样的图景会遮蔽语言里的缺失、模糊和欲望维度。萨斯能说明施瑞伯的言说为何带有逻辑悖论，却没有说明：主体为什么要在此刻以这种方式组织妄想？精神病与神经症在欲望层面究竟差在哪里？在她看来，这正是维特根斯坦式路径的盲点。

## 模块三：语法无误却无法对话——临床案例、精神自动症与白色精神病

**Q:** 墨西哥流浪汉的临床案例说明了什么？
**A:** 小柏引用切尔马克在圣安娜的个案：病人的语法极其整洁，回答也都“对题”，但始终只停在字面与事实层面。医生问“你是在什么情况下来到这里的”，他回答“坐飞机来的”；问墨西哥对他意味着什么，他说“是另一个阶段”；问有没有女朋友，他只说“有过几个好朋友”。这些回答并非语法错误，而是系统性地回避主观经验、历史、自我解释与他者欲望的维度。

**Q:** 为什么这种语言会让人感到“可读却不可理解”？
**A:** 因为问题根本不在共享语法失效，而在主体如何使用语言。小柏借拉康的框架说，这类言说能服从 `S2` 层面的语法和连贯，却仿佛没有一个可定位的主体性起点；他知道怎样组成句子，却不把自己放入言说之中。于是语言成了封闭自己的工具：一旦他者通过提问试图“拿走一点什么”，主体就用极其精密、无修辞的语法把自己重新封住。

**Q:** 小柏为什么随后转向德·克莱朗博的“精神自动症”？
**A:** 因为她要借德·克莱朗博把因果顺序彻底倒过来。按她的理解，精神病主体首先遭遇的并不是一种高反思的唯我论建构，而是一种更原初的断裂：主体以被动姿态碰上某种“原物”式的、无逻辑的不可理解性，原先维持其生命平衡的象征构造在这里失效了，留下的是迷茫、困惑以及身体层面的僵化、停顿等效应。德·克莱朗博所谓“精神自动症”对她来说，重要的不是罗列若干怪异现象，而是说明这种断裂怎样先一步发生。妄想因此不是起点，而是在这之后才被主体调用出来的次级解释性构建：主体重新开始整合，借助自己的文化、教育史和既有材料，把遭遇到的东西编织成一个叙事，哪怕这个叙事再离奇，也是在为无法直接承受的断裂做补缀。这样看，小柏批评萨斯的重点就很明确了：萨斯把问题处理得太像一种哲学化的意识建构，却没有抓到精神病首先是主体遭遇不可处理之物、随后才通过妄想去填补裂口、重新把自己勉强注册回世界的过程；这也是为什么他的解释更容易卡在施瑞伯这类妄想建构丰富的案例上，而难以覆盖“白色精神病”。

## 模块四：粉色海猡的第一命题——精神分裂理论背后的政治意识形态

**Q:** 粉色海猡如何概括自己报告的两个命题？
**A:** 她把问题压缩成两句：第一，每一种主流的精神分裂理论，都对应着一种关于“什么才算人”的政治意识形态；第二，那些被拿来验证理论的临床症状，并不是纯粹自然地摆在眼前，而是在特定制度装置中被生产、被塑形出来的。

**Q:** 她为什么说萨斯其实是 Laing 的“保守化版本”？
**A:** 因为两人都保留了一个人之为人的核心机构。Laing 把精神分裂理解为主体在无法生存的社会关系中发明出的防御策略，背后仍有一个被威胁的“真实自我”；萨斯虽然换用了胡塞尔、维特根斯坦与 hyperreflexivity 这样的说法，但仍假定精神病人有一个内省的理性中枢在加工经验。区别在于，Laing 更强调异化社会和物化关系的压迫，萨斯则把压力转译成现代主义式的怀疑、反思和自我坍塌。

**Q:** 这会带来什么理论后果？
**A:** 粉色海猡的回答是：精神分裂的致病与发病就会被从政治和物质环境中抽离出来，仿佛病人的受苦与机构安排、护理程序、预算和行政权力的分配无关。她因此主张把精神病理论放回意识形态批判中看：每个时代定义疯狂的方式，都和它对“正常人”的定义捆绑在一起。

## 模块五：机构如何生产症状——从施瑞伯到拉博德诊所

**Q:** 粉色海猡如何重新解释施瑞伯的“无能感”和“全能感”？
**A:** 她把施瑞伯放回松嫩施泰因疗养院的微观政治里读：医生、护士、行政与病人之间等级森严，病人身体被纳入严格的时间表、空间隔离、观察记录和治疗程序。这样一来，“我什么都做不了”的无能感不是纯主观幻象，而是现实处境的身体性真相；同时，“整个世界都围着我转”的全能感也不是平白生成的，因为病人确实被安置成整个装置的中心焦点，一举一动都被观察、书写和阐释。

**Q:** 她为何强调精神病主体并非“没有移情”，而是拥有群体性的移情？
**A:** 粉色海猡借托斯凯勒和后来的机构治疗理论说，精神病人的转移并不是缺失，而是被离散到众多成员和场所上，形成一种群体性的 transferential constellation。问题不在于他不能投注，而在于投注无法稳定地锁定在单一客体上。传统精神病院把这种星座式关系压扁为被管理的对象关系，症状就会越来越朝向被监视、被操控、被侵入的样态凝固。

**Q:** 拉博德诊所的“导演病人”案例，为什么与施瑞伯截然不同？
**A:** 因为在拉博德，俱乐部与“网格”工作法打破了医生、护士和病人的僵硬边界。病人可以在群体活动、轮换劳动和开放空间中，把自己的幻想转化为具体的媒介实践。那个想拍电影的病人，不再只能把全能感向内挤压成妄想，而是能够真的把医护编入剧本、拍出作品。粉色海猡由此得出结论：精神分裂的症状样态高度依赖场所氛围；开放、差异化、允许新能指生成的诊所，并不是在“容忍疯子”，而是在给主体寻找新的表征位置。

## 模块六：陈劲骁的评论——晚期拉康、语言之墙与翻译动机

**Q:** 陈劲骁怎样理解妄想、精神分裂与晚期拉康之间的关系？
**A:** 他先指出，这本书标题把“妄想”和“精神分裂”并列在施瑞伯身上，本身就说明二者不能被僵硬地分成互斥范畴。接着他把问题推向晚期拉康：如果拉康说“人人皆疯狂”或“人人皆妄想”，那重点就不再是给谁贴上病名，而是要重新理解语言、意义与主体之间的裂缝。

**Q:** 他为什么说晚期拉康不是“发疯了”，而是在逼近语言极限？
**A:** 陈劲骁认为，拉康越往后越不再相信象征秩序能穷尽真相，所以才会从语言哲学式的严整表述，转向拓扑图、声音、字母和各种看似怪异的造词。所谓“语言之墙”不是修辞夸张，而是在说人和人并不能通过符号透明互通；我们总是在回音、误听和材料性的声音中互动。这样一来，精神病的“不可理解”也就不该被当作某种单纯等着翻译的隐藏密码，而更接近一种实在层面的回响。

**Q:** 他为什么会去翻译《妄想的悖论》？
**A:** 陈劲骁坦承，起点反而是自己的偏见。初到法国时，他感到当地拉康派像宗教共同体，和自己此前接受的教科书式、象征性取向训练发生了断裂，因此一度把精神分析本身看成某种“妄想”。也正因为他对哲学感兴趣，又被书中维特根斯坦与精神分析的连接所吸引，才会进入这个题目。后来他逐渐修正了早先把“科学/宗教”二分法硬套到精神分析上的冲动。

## 模块七：问答延伸——圣状、妄想隐喻、政治与零度分析

**Q:** 观众1追问：既然拉康高度评价乔伊斯的圣状，为什么露西亚仍然精神病性发作？
**A:** 陈劲骁的回答是，乔伊斯通过书写、命名和关系建构，为自己做出了稳定化的增补，但这种增补不是一个可被简单传递给下一代的“父之名”。小柏进一步补充：露西亚这个名字本身就像对父亲缺陷的一种补位，她也有舞蹈和诗写作等潜在增补资源，但当时的治疗环境和时代条件并没有帮助她把这些资源稳定成自己的位置。两人的共同点是都强调：乔伊斯的稳固不能自动覆盖到女儿身上。

**Q:** 观众2追问：妄想隐喻与父亲隐喻有什么区别？
**A:** 小柏和陈劲骁都把两者严格分开。父亲隐喻是拉康用来描述神经症主体发生学的结构概念，它为“缺失”命名，使主体面对他者欲望时不至于被焦虑彻底淹没；妄想隐喻则是精神病主体在精神自动症、被害体验或言语入侵已经发生之后，才发展出的补救性 solution。施瑞伯把自己定位为“上帝的女人”，就是一种典型的妄想性增补：它荒诞，却能稳定主体。

**Q:** 观众1追问：精神分析为什么总会和政治纠缠在一起？
**A:** 粉色海猡给了一个制度史回答：1968 年之后，法国大规模公共资助推动了机构分析、反精神病学和相关理论实验，等到政治风向转变，很多工作也随之衰退。小柏则补充说，四大话语、第二波女性主义、反俄狄浦斯、生命政治等问题本来就在同一历史语境里生成，精神分析不可能置身于这些“政治无意识”之外。

**Q:** 观众1最后问到的“零度分析”，和罗兰·巴特的“零度写作”是什么关系？
**A:** 陈劲骁明确说，拉康晚期的“零度”确实借用了巴特的概念。它意味着分析家尽可能撤回以理论、解释和术语去占满现场的冲动，把位置退向一个更接近“零”的地方，让来访者的话语材料、声音双关和奇异性自己浮现出来。这种姿态与弗洛伊德式释梦形成明显反差，但陈劲骁也强调，这不是普遍处方；它有先锋性，也有局限，必须放回具体临床情境中判断。
