# 讲座笔记：新年漫谈：穿越幻像是可能的吗？

## 模块一：利奥塔、力比多经济学与欲望主体传统

**Q:** 利奥塔的《力比多经济学》大致在处理什么问题，为什么嘉泰会把它描述成一种“形式主义化”的工作？（行 52–57，约为原文70%字数）
**A:** 嘉泰先把这本书定位成他准备翻译的利奥塔代表作之一。他强调，这本书虽然叫《力比多经济学》，但并不是按经济学叙事去讲问题，而是把力比多的流动、投注、升华与下降等过程，放进一种法国当代形式主义研究的传统里，做成高度抽象的研究。所谓“经济学”，更像是一种冷峻的类比框架，而不是学科归属。与此同时，利奥塔又会把古罗马的戏剧神学、狂欢节这类历史材料带进来，让这种本来很“无情”的抽象叙述混入节庆、神秘和欲望释放的色彩；嘉泰觉得，这种把冷的形式化工作和热的历史材料缝在一起的写法，正是它特别有意思的地方。

**Q:** 《力比多经济学》与精神分析究竟是什么关系？（行 59–64，约为原文75%字数）
**A:** 面对诡谲子“它到底是精神分析还是经济学”的追问，嘉泰的回答很明确：利奥塔本人并不把这本书当作精神分析著作。若硬要说关系，它主要只借用了“力比多”这个关键概念，却没有沿着精神分析解析欲望的那条路走下去。按嘉泰的说法，利奥塔更像把欲望朝一种自然主义解放的方向推，而这恰好和拉康在《精神分析伦理学》里批评的方向接近，等于和拉康走上了相反的道路。也正因为它只拿走一个概念、却不接受整套分析框架，所以它和精神分析既有关联，又保持了相当大的距离。

**Q:** 如果它并不真正属于精神分析，那么它借“经济学”这个说法究竟在强调什么？（行 66–80，约为原文65%字数）
**A:** 嘉泰进一步解释说，利奥塔确实会调动一些经济史材料，比如殖民时代的白银货币体系，但目的不是把自己放进某个经济学传统，而是借经济学说明一个隐喻性的机制。他举的例子是码头上的水手拿银币去酒馆消费：这枚银币看起来只是中性的交换媒介，实际上却浓缩了水手数年海上讨生活的风险、冲动和力比多活动。换言之，货币这种“死物”之所以能完成交换，恰恰因为大量强烈的驱力在其中被符号化、被相互抵消，最后以中性的形式出现。诡谲子把这一步概括成“被符号化、货币化之后进入一种 equilibrium 状态”，嘉泰明确点头。于是，“经济学”在这里不是学科归类，而是指驱力如何在交换中被压缩、平衡和中性化。

**Q:** 为什么两位讲者都认为中文语境对利奥塔的理解仍然过窄，他的方法特点又在哪里？（行 82–100，约为原文55%字数）
**A:** 这一段里，诡谲子先说，最近几年利奥塔在中文世界确实重新受关注，但翻译仍然太少，大家往往只记得《后现代状况》、宏大叙事之类的标签，对别的路径几乎没有展开。他还提到自己在比利时遇到研究利奥塔的学生，对方特别强调利奥塔的现象学维度，这恰恰是国内几乎没人讨论的线索。嘉泰顺着这个判断补充说，这种理解“太少、太不够了”。在他看来，利奥塔也在做把精神分析冲动乃至某种神秘主义材料加以形式主义化的工作，只是他的写法和巴迪欧不同。巴迪欧不仅研究方式形式主义，连行文本身都形式主义到晦涩、逼得读者回头补群论和高阶范畴论；相比之下，利奥塔虽然也抽象，但更像一个“正统哲学家”，叙述风格没那么硬。

**Q:** 应该把利奥塔放回法国哲学的哪条线索里理解，他又如何接续到“欲望主体”的传统？（行 102–112，约为原文80%字数）
**A:** 对“利奥塔属于哪条法国哲学脉络”的问题，嘉泰给出的定位是：他更接近一条把爱欲当作本体论问题来处理的法国哲学路线。这里的关键不在于他是否使用了“力比多”这个词，而在于他赋予欲望、爱欲一种近乎本体论的地位。但嘉泰也再次强调，这不是精神分析范式内部的工作，而更像让-吕克·马里翁那种“情欲现象学”的方向。诡谲子随后把这条线索接到了自己熟悉的“欲望主体”传统上，嘉泰立即认可，并把它扩大成一个更大的判断：整个 20 世纪法国哲学的很多重要工作，都在把欲望或爱欲抬到非常根本的位置。

## 模块二：嘉泰的学术路径与“大师幻想”

**Q:** 嘉泰为什么会直接把博士 proposal 投向巴迪欧本人，后来又是怎样转到齐泽克门下的？（行 115–139，约为原文60%字数）
**A:** 嘉泰说，他最早并没有打算直接去找巴迪欧本人，而是先找了法国和澳大利亚几位巴迪欧学生，包括 Justin Clemens 这一支，想看看这些直系门生能不能接住自己的 proposal。结果他发现，对方虽然“学了一些东西”，但一谈到范畴论和巴迪欧核心的数学哲学问题，就明显接不住，也不愿为超出自己能力范围的课题负责。于是他才反过来想：既然门生都接不住，那就只能直接找巴迪欧本人。他把 proposal 投到当时仍在欧洲高等学院、主持勒内·笛卡尔教席的巴迪欧那里，巴迪欧则以年纪太大、准备退休为由婉拒，并把问题转给学校。后来在校长协调下，这个项目被推给齐泽克。齐泽克起初并不情愿，因为课题与巴迪欧相关度更高；嘉泰则强调自己的研究也涉及晚期拉康拓扑学，以及一种绝对形式主义的分析工作，所以齐泽克多少还能“接住一点”。最后齐泽克算是勉强同意，但嘉泰也直说，他后来的指导状态基本上还是“甩手掌柜”。

**Q:** 围绕“跟大师读书”这件事，嘉泰如何处理其中的幻想成分与现实成分？（行 141–153，约为原文55%字数）
**A:** 诡谲子先用玩笑把这件事拆穿了一半：学习本质上还是自学，所以就算没跟齐泽克读书，也可以“幻想”。嘉泰并不否认这种幻想成分。他说，自己在 B 站上传的很多内容其实是长篇学术演讲，本来并不好看，但观众之所以愿意“买账”，显然和“跟大师读书”的想象有关。与此同时，他也没有把这件事完全贬成空壳，因为齐泽克当然仍然是大师。只是如果把这个关系看得太独一无二就夸张了，毕竟齐泽克这辈子带过的学生很多，“多我一个不多，少我一个不少”。换句话说，大师光环确实真实存在，也确实会构成幻想的支撑，但它既不是全部，也不是决定性的救赎。

## 模块三：幻想的必要性与“穿越幻象”的边界

**Q:** 人为什么总得保留某种幻想，幻想在结构上究竟起什么作用？（行 155–180，约为原文60%字数）
**A:** 当诡谲子把话题正式转到 fantasy 时，嘉泰先给了一个干脆的结论：人总得有幻想。如果一个人不断刀刃向内，把自己的幻想全部割掉、阉割掉，最后很可能会掉进虚无主义和抑郁状态，严重时甚至连起床刷牙都懒得做。接着他有意不重复齐泽克那套“幻想参与现实再组织”的常见说法，而是从更结构主义、形式主义的角度重讲一遍。他说，符号本质上是离散的、最小的形而上学形式，不管放大还是缩小都不改变其意义；而幻想的功能，就是用某种假定的、不充分的形式，把原本接不起来的离散符号在局部连起来，使其呈现出连续运作的状态。也正因此，幻想不是可有可无的错觉，而是让很多东西得以运转的结构性条件。

**Q:** 一个人如果沉浸在幻想中被其裹挟，为什么这种状态会被嘉泰判断为不可持续？（行 182–194，约为原文70%字数）
**A:** 诡谲子随即把问题推进到反面，拿《包法利夫人》和《情感教育》里“把虚构当真”的人物做例子，追问如果人彻底沉浸在幻想里怎么办。嘉泰先接住福楼拜的典故，提到那句著名的话“她想死，也想去巴黎”，又半开玩笑地说自己也有过“包法利夫人式”的前女友经验。真正的回答则是：这种状态未必需要先从道德上判断对错，因为它会以自身的运动形式给出判断。换言之，靠幻想维持的力比多投注如果不能完成自身的再生产和循环，那它就注定不可持续；等到这种不可持续性成熟到一定程度，它一定会自己暴露出来，而不以任何人的意志为转移。所以崩塌不是因为旁人不够包容，也不是当事人不愿意努力，而是因为这种生活结构本身就无法长久运转。

**Q:** 诡谲子提出的“穿越幻象”在艺术社会学的语境里是什么意思，嘉泰又如何修正这个说法？（行 196–211，约为原文70%字数）
**A:** 这一问里，诡谲子先把自己的想法摆出来：也许二十出头时人确实需要很多 fantasy 拽着往前走，但等到某个年龄，他更想在教学里引导学生“穿越幻象”。他给出的具体场景是未来给美院新生上艺术社会学，核心区分是美学处理消费者的感受与接受，而艺术社会学处理生产机制、生产方式和生产关系，属于一种“生产者的美学”。嘉泰并没有否认这个方向，而是对“穿越”做了更细的修正。他说，消费本身不可持续，纯粹的生产反而可以持续；所以更好的状态不是把幻想彻底废掉，而是让幻想在生产过程中发挥局部效用。按照他前面的结构主义解释，就是两个象征暂时连不起来时，允许幻想在局部把它们接上，但在整体上又清楚知道它毕竟是幻想。对嘉泰而言，这种“知道它是幻想，却仍有策略地运用它”的状态，才是比较好的“穿越幻象”。

**Q:** 面对刚入学的年轻学生，是否应该一上来就帮助他们“穿越幻象”？（行 213–224，约为原文75%字数）
**A:** 对诡谲子“那我是不是该给美院大一新生先讲这些”的追问，嘉泰的态度是谨慎。他直说，这样的敲打可能太狠，必须分人看。因为有一类学生是一路在考试体制里被压抑着走上来的，他们的欲望长期受压，这时候出现的幻想并不一定是恶性的，反而更像对其他伤害的一种代偿或反扑。嘉泰承认，他们终究有必须接触真实的时刻，但教师更合适的做法，是在过程中给出征兆、预示和提醒，而不是直接动手切割幻想。最后他把这个判断落回自己的临床经验：他做过相当长时间的精神分析临床，因此在面对来访者时形成了一种伦理态度，那就是直接动手打碎一个人的幻想，是非常危险的。

**Q:** 为什么诡谲子仍坚持要更早揭露生产关系，而嘉泰提醒这件事若处理失当会把人“敲蒙”？（行 226–259，约为原文70%字数）
**A:** 这里两人的分歧被说得比较清楚。诡谲子先承认，从精神分析伦理出发，嘉泰的顾虑成立；但如果换成当代艺术从业者和马克思主义者的角度，他反而觉得应当更早、更直接地揭露生产关系和生产机制中的剥削与矛盾，比如点破那些让人“用爱发电”的美术馆机制。嘉泰的回应不是简单反对，而是把问题转到教育场景的具体条件上：教师当然要讲普遍性的道理，但面对学生这种既有个人性、又折射一代人命运的问题时，不能假装自己能为他们的人生负责到底。老师不给学生出钱，也不能一路提着他们考研考博；更重要的是，学生之间的承受能力和理解速度确实不同。诡谲子随后举了文学理论和批判理论的例子，说很多人一接触理论就失去阅读兴趣，甚至因既有世界图景崩塌而抑郁。嘉泰接着把危险讲透：有些人被批判“敲”了一下，并不是被敲醒，而是被敲蒙了。他们没有进入新的精神阶段，只学会了拿批判当棍子去敲别人，最后变得犬儒、刻薄、好面子、爱批评，却不承担任何东西。这也是他对当下更年轻学生最担心的地方。

## 模块四：从被“敲蒙”到争夺公共领域

**Q:** 当批判没有把人敲醒，反而把人推向犬儒、刻薄和姿态化时，问题出在哪里？（行 261–275，约为原文80%字数）
**A:** 诡谲子把嘉泰前面的描述直接和“往左”的年轻姿态联系起来，还追问是不是很多初高中生已有这种味道。嘉泰的判断非常直接：这绝对不是好事。问题不只是他们太会否定，而是他们并没有把自己的批判做成能穿透个别性的东西。按他的说法，如果一种批评根本进不了公共领域，它就没有意义；公共领域的选择本身，就是最好的批判。这里的“公共领域”也不是简单等于当下大众立刻买账，而是说一种观念必须有穿透特殊性、成为普遍性共识的能力。若只停在私下摆姿态、互相刺伤、彼此比谁更尖锐，那批判就会退化成犬儒风格，而不能成为真正有效的工作。

**Q:** 为什么嘉泰说“幻想是不彻底的真实”，并据此反对那种把事情悬置在中间的生活态度？（行 278–316，约为原文65%字数）
**A:** 为了把这个问题说得更具体，诡谲子先举了两个例子：一是《黑客帝国》式的红药丸/蓝药丸选择，二是自己认识的一个文学系学生，被阿尔都塞、福柯和后现代理论“讲蒙”以后，研究生转去古代文学、古文字学，像是在寻找新的确定性。嘉泰承认这个跨度很大，但他借此提出一个更核心的判断：幻想并不是与真实分离的另一世界，它只是“不彻底的真实”。也就是说，幻想之所以成为幻想，不是因为它脱离现实，而是因为它悬停在一种真实与另一种真实之间，既没有彻底成为第一种，也没有彻底成为第二种。由此他非常反感那种“想不通就不想，日子照过”的顺其自然态度，因为被悬置的问题不会消失，只会在潜意识层面把责任转嫁给其他人，让所有人在不自知中变得暴躁。对他来说，最低限度的伦理要求不是把事情做得多伟大，而是不要让它永远卡在中间，至少要让它真正成为什么东西，而不是永远“什么都不是”。

**Q:** 面对学术履历主义、庸俗普遍性和资本主义现实，两位讲者为什么都认为不能退回犬儒，而要进入公共领域争夺真正的普遍性？（行 321–360，约为原文65%字数）
**A:** 这一段里，两人的立场越来越趋同。嘉泰先说，自己过去对自己和别人都很苛刻，现在多少看开一点、也更能包容别人；诡谲子则把话题拉到现实学界，抱怨那些发了几十篇 C 刊、却连研究兴趣都说不清的教授，认为这是一种对自我毫无要求的状态。嘉泰马上把这种履历主义命名为一种被精心包装的布尔乔亚式投机，是把学术当作资本主义工作的做法，只是社会偏偏吃这一套。即便如此，两人都反对退回生气、躺平和犬儒。嘉泰说，既然看不顺眼，就更应该上去写文章、卷进去，把“尸位素餐的人”卷下去，讲出更好的东西。诡谲子也修正了自己早先那种把波德莱尔式闲逛理解为反资本主义的想法，转而主张真正的反资本主义是“深入敌后”，进入现实去真刀真枪地做事。嘉泰最后把这个态度概括成一句更理论化的话：不是因为现有公共领域建构出的是庸俗的普遍性，我们就可以逃离公共领域；相反，必须进入其中，去取代它、建立真正的普遍性。这种关系甚至像主奴辩证法，不经历它的奴役，就无法争取成为它的主人。

## 模块五：欧洲高等学院、当代理论现场与代际接续

**Q:** 嘉泰如何描述欧洲高等学院的办学方式、地理环境和整体学术气质？（行 363–379，约为原文70%字数）
**A:** 回到自己在欧洲高等学院的经历时，嘉泰先强调，这所学校和传统大学最大的不同，是它严格说来“不授课”，而是以 seminar 和研讨的方式运作。它的理想不是教师单向讲授，而是持续的对话。嘉泰把这种办学气质追溯到学校的来历：五月风暴之后，一批左翼哲学家离开传统大学体制，自己凑资源、拉赞助，办出了这样一个机构，所以它天然带着一种反传统、去思辨的气质。地理环境也强化了这种风格。学校不在城市里，而在瑞士瓦莱州萨斯费的阿尔卑斯山小镇，平时更像滑雪胜地而不是都市校园。嘉泰用略带戏谑的语气形容那里像一群“哲学老灯”围着小木屋和火炉讨论问题，颇有一种海德格尔晚年山中隐居式的氛围。

**Q:** 近几年欧洲高等学院的 seminar 大致围绕哪些人物和议题展开，嘉泰对这些前沿讨论有什么印象？（行 382–421，约为原文65%字数）
**A:** 被问到 seminar 的大致方向时，嘉泰先承认课表非常杂，很难整齐分类，所以只能挑印象深的说。第一条线索是齐泽克。嘉泰提到，齐泽克近年围绕 `Towards a Liberal Fascism` 这个主题讲过 seminar，重点不是重复旧译本里那些大家熟悉的说法，而是试图把握 2020 年代这种“像死亡腐尸一样”、缺乏差异和变化的世界状态。第二条线索是 Judith Butler。嘉泰说，Butler 也是学校常客，过去他总觉得她在政治哲学、性别、少数族裔、犹太视角以及交叉性等问题上相对温和，还相信很多矛盾可以调和；但最近几年她在那里的讲法明显更残酷了，更强调矛盾的不可调和。除此之外，嘉泰还提到学校会邀请亚非拉文学家、被以色列驱逐的导演和作家 Udi Aloni，以及 Frank Ruda、校长 Christopher Fynsk 等人。嘉泰总结说，目前学校牌面主要靠齐泽克、阿甘本和朱迪斯·巴特勒这批老牌哲学家撑着，新一代还在接班途中，更像很优秀的研究者，还没有完全长成现象级的大哲学家。

**Q:** 当老一代大哲学家逐渐退场、既有理论又显得滞后时，两位讲者如何理解新一代学者的任务？（行 422–474，约为原文65%字数）
**A:** 从“大佬们都远去了，我们得顶上”这句玩笑开始，两人把话题转成了更严肃的代际判断。诡谲子认为，任何思想真正长成，都必须和现实发生猛烈碰撞；只是过去全球化时代相对平静，导致许多课堂和理论工作显得过于无聊、过于过时，甚至跟不上时代，更别说解释时代。嘉泰对此基本同意，但他的回答更克制一些：如果真要说当下还能做什么，也许只能先替上一辈学者解决他们体系里遗留下来、尚未彻底解决的问题；更宏观的设想并非想不到，而是社会太疲惫、太倦怠，很难对它作出足够有活力的回应。诡谲子最后给出的方向，则是继续吸收和转化 1960 年代之后的理论遗产，把这些理论重新语境化，搞清它们是如何生成的、有什么问题；与此同时，学者不能只做继承者，也要发展出自己真正关心的问题，并据此综合、总结、创造新的理论去回应现实。两人的共同点在于，他们都不再把希望寄托在大师个人身上，而是把任务理解为一边清理旧理论的欠账，一边为新的现实准备自己的问题意识。

## L: What's Left

- [书目] 利奥塔，《力比多经济学》— 嘉泰只交代了其“形式主义化”与“经济学隐喻”的骨架，文本内部结构仍未细读
- [书目] 利奥塔，《康德的崇高分析论讲稿》— 被提到即将出版，可作为补足利奥塔现象学线索的入口
- [书目] 齐泽克、Frank Ruda，《Towards a Liberal Fascism》— 嘉泰明确说其中有不少“劲爆”内容，且未能翻成中文
- [书目] 福楼拜，《包法利夫人》与《情感教育》— 两部小说被拿来说明幻想的不可持续性，但没有展开比较

- [脉络] **利奥塔—拉康—马里翁** — 欲望/爱欲如何在精神分析、现象学和本体论之间转位，这期只给了轮廓
- [脉络] **巴迪欧—齐泽克—晚期拉康拓扑学** — 嘉泰的博士课题背景被点到，但具体理论问题没有展开
- [脉络] **欧洲高等学院的老一代与新一代** — 谁在接班、为什么暂未出现新的“现象级”理论家，还值得进一步梳理

- [概念] **幻想 / 穿越幻象** — 嘉泰给出了结构主义说明，但它与齐泽克常说的“幻想组织现实”之间的关系还可继续拆开
- [概念] **公共领域 / 普遍性** — 怎样把批判从姿态变成能进入公共领域的普遍性，本期只给出方向，没有给出更细的路径
- [概念] **生产者的美学** — 诡谲子用它区分美学与艺术社会学，但还没有展开更完整的课程框架

- [话题] 教学里何时该敲打学生幻想、何时只该给征兆 — 两人都承认必须分人分场合，但没有提出更具体的判准
- [话题] 既有理论“过时”以后如何回应新现实 — 两人都承认自己没有完整答案，更多停留在自我要求与准备阶段
- [话题] 中文语境为何长期只把利奥塔等同于“宏大叙事批判” — 节目点出了问题，但没有继续分析其知识史原因
